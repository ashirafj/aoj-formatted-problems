<H1>Distance</H1><br>

<p>
  Write a program which calculates the distance between two points P1(x1, y1) and P2(x2, y2).
</p>


<H2>Input</H2>

<p>
  Four real numbers x1, y1, x2 and y2 are given in a line.
</p>


<H2>Output</H2>

<p>
  Print the distance in real number. The output should not contain an absolute error greater than 10<sup>-4</sup>.
</p>

<H2>Sample Input</H2>

<pre>
0 0 1 1
</pre>

<H2>Sample Output</H2>

<pre>
1.41421356
</pre>
