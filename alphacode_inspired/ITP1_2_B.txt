CORRECT SOLUTION
Write a program which reads three integers $a$, $b$ and $c$, and prints "Yes" if $a < b < c$, otherwise "No".

Input

Three integers $a$, $b$ and $c$ separated by a single space are given in a line.

  * $0 \leq a, b, c \leq 100$

Output

Print "Yes" or "No" in a line.

Examples

Input


1 3 8


Output


Yes


Input


3 8 1


Output


No
