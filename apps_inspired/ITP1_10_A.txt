Write a program which calculates the distance between two points P1($x_1$, $y_1$) and P2($x_2$, $y_2$).

-----Input-----

Four real numbers $x_1$, $y_1$, $x_2$ and $y_2$ are given in a line.

-----Output-----

Print the distance in real number. The output should not contain an absolute error greater than $10^{-4}$.

-----Example-----
Input
0 0 1 1

Output
1.41421356
